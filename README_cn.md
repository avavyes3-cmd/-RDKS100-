# 基于 RDKS100 的森林火灾多模态巡检系统

## 项目简介

针对森林火灾监测中传统人工巡检效率低、覆盖范围有限、单一传感器易受环境干扰等问题，本文设计并实现了一套基于地平线 RDKS100 嵌入式平台的森林火灾多模态巡检系统。系统融合可见光与热红外双光感知、激光雷达 SLAM 自主导航、边缘 AI 实时推理等关键技术，构建了**感知—配准—识别—巡检一体化闭环**方案。

## 应用场景

- 森林防火日常巡检与早期火情预警
- 大面积林区自主覆盖式巡查
- 复杂地形与恶劣天气条件下的火灾监测
- 火灾高发期的无人化值守

## 技术架构

### 感知层
- **可见光相机**：采集高分辨率 RGB 图像，用于场景理解与烟雾识别
- **热红外相机**：感知温度异常区域，实现火点探测与高温预警
- **激光雷达（LiDAR）**：获取三维点云数据，构建环境地图并实现自主导航

### 配准层
- 可见光与热红外图像的多模态配准，实现像素级对齐
- 点云与图像的时空同步与融合

### 识别层
- 基于地平线 BPU 的边缘 AI 推理引擎部署
- 实时烟雾检测、火焰识别、温度异常判别
- 多传感器融合决策，降低误报率

### 巡检层
- 激光 SLAM 自主建图与路径规划
- 基于 RDKS100 的嵌入式板载控制
- 巡检任务自动执行与异常上报

## 快速开始

1. 下载并解压分卷压缩包：
   - `s100_non_registration_code_20260707_send.part1.rar`
   - `s100_non_registration_code_20260707_send.part2.rar`

2. 使用解压后的工程文件，在地平线 RDKS100 开发环境中编译部署

3. 连接各传感器模块，上电启动巡检系统
